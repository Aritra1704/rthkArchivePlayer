RTHK Archive Player
===================

Introduction
------------

RTHK Archive Player is an Android application that can play the stream of RTHK's archive.
It crawls [RTHK website](http://www.rthk.hk/) to search for avaliable archives.
It supports bookmark and downlaod of the archives.
